# LazyDash 3.0

Olá, esse é meu projeto para Criar Dashboards de forma fácil e prática, assim como a tecnologia deve ser. Como ele você pode ser capaz importar sua planilha e gerar automaticamente algumas opções de dashboards com IA para sua análise de dados, da mesma forma, você também pode editar eles após criados ou criar do zero do seu próprio jeito.

Outra função interessante do projeto é a capacidade de gerar análises sobre seus dados, como insights, avaliações e recomendações sobre o tema analisado.

## Informações importantes

O projeto está usando o python 3.12.7 para melhor estabilidade, e todas as bibliotecas estão no "requirements.txt"

## Especificações

- Python 3.12.7
- Django
